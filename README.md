# oaqjp-final-project-emb-ai

## Emotion Detection Application using IBM Watson NLP

This project is an Emotion Detection web application developed using Python and Flask.
It analyzes user-provided text and detects emotions such as:

- Joy
- Sadness
- Anger
- Fear
- Disgust

The application processes the input text, calculates emotion scores, and determines the dominant emotion.

## Features

- Emotion detection from text input
- Error handling for invalid or blank input
- Flask-based web interface
- Modular Python package structure
- Static code analysis compliance (pylint)

## Technologies Used

- Python 3
- Flask
- IBM Watson NLP API (simulated implementation)
- Pylint for code quality

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the server:
   python3 final_project/8a_server_modified.py

3. Open browser:
   http://localhost:5001
